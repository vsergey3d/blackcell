B.Graph

Contains scene graph nodes and utilities.

ENUMERATIONS:

Order

Graph traverse order.

{number} PRE = 1

Pre-order (depth-first) traversal.

{number} POST = 2

Post-order (depth-first) traversal.

FUNCTIONS:

makeNode()

Makes a node.

RETURNS:

{B.Graph.Node}

makeLocator()

Makes a locator (transformed location in 3D-space).

RETURNS:

{B.Graph.Locator}

makeVisual()

Makes a visual (transformed mesh + material).

RETURNS:

{B.Graph.Visual}

makeCamera()

Makes a camera.

RETURNS:

{B.Graph.Camera}

makeLight()

Makes a light.

RETURNS:

{B.Graph.Light}
